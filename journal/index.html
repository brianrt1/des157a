Journal 03

An image can be used several different ways when designing for the web. Imagery to web designers is a chance to incorporate a visual language through using photographs as design elements. There are some techniques and things to avoid when using images however. Images can communicate ideas and emotions, that means that they are treated with a high level of priority in the design process. Choosing your images early on is important, building the design around the ideal imagery intentionally for the purpose. In the Microdot example by Standard Projects, Imagery is a dominant element which overall creates an immersive feeling. An animated gradient fills the background, while an image gallery fades in as you scroll down. This has a dynamic and image-centric feeling, that is more taxing on the browser. Additionally, working with many images makes creating a responsive design more challenging. Choosing the right images begins with a plan for visual styling and overall narrative of your design.

Journal 02

The misuse of overlays in web design can lead to poor user experiences, where the main task is being hindered by distracting pop-ups. Overlays appear to be a convenient choice to create custom, practical forms in order to deliver or capture information without breaking your webpage. This avoids having to rely on browser alerts and prompts to display information and get feedback. However, these overlays are often used improperly by making websites more cumbersome to navigate. This happens for several reasons, one major reason is related to the size of the overlay which can be too large. When not optimized for mobile views, the overlay will likely cause problems. An overlay can be easily misused, misuse of overlays impacts both accessibly and user experience, which negatively impacts usability. Consider if and overlay is appropriate for the main task, it can often be unnecessary. When used properly, it can make users feel more secure and invested in the content. Overlays are a creative medium to present and collect information, adding another layer of design language and interactivity.
